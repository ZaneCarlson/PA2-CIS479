# PA2-CIS479

Run this program with no parameters and follow these steps.

1. input how many samples you want to run
2. input two sets, the first will be evidence vars. The second will be query vars. Query vars are only going to be checked as true.
3. example input "[< A, f >< B, t>][J, M]"

it will output the exact probability of your query followed by the results of each sampling type.
